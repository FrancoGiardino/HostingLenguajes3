¿Que es HTML? HTML (HyperText Markup Language) es el componente mas basico del web. Define el significado y la estructura del contenido web. Ademas HTML, generalemente se utilizan otras tecnologias para describir la apariencia/ presentacion de una pagina web (CSS) o la funcionalidad/comportamiento (JavaScript). La version usada actualmente es HTML 5.
